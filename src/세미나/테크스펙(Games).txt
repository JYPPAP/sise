요약 (Summary)
Container 에서 사용자가 게임을 선택했을 때 출력되는 페이지
헤더에는 선택한 게임 명, 바로 아래의 서브 타이틀에서는 기준이 출력된다.
서버, 평균가, 등락아이콘이 출력된다.
서버를 선택하면 서버별 15일간의 등락폭이 출력되는 페이지로 이동한다.

계획 (Plan)
router 의 params를 받아오고, gamelist에서 게임명에 따른 게임도 가져온다.
서버목록 또한 가져온다. 
  > 가져온 값에서 당일 - 전일을 비교해 아이콘을 출력한다.
아이템 부분을 출력한다.
item 부분을 함수로 만들고, col1, col2, col3, col4, col5 등으로 나눠서 관리한다.
세부 스타일은 ID 값을 이용하고, 공통적인 스타일은 class로 관리한다.
서버를 선택하면 Link를 이용해 "/games/:gameName/servers/serverName 페이지로 이동한다.
